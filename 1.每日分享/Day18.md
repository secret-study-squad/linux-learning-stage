1. **vim与xshell的配置文件修改**

   ​		在xshell与vim的实际使用中，会发现默认设置极其难用，但是有许多扩展命令可以给我们使用，但是每次命令生效之后只在此次连接中有效，总不能一直保持电脑为开机状态，所以可以通过修改配置文件来使效果永久生效，这是因为每次xshell与vim运行前都会先执行它们自己的配置文件，并且经过我的使用测试，每个用户之前的配置文件都是独立的，这两个工具的配置文件都在每个用户的家目录之下。

   ```c
   // 例如：我的用户名gzh的xshell与vim的配置文件的绝对路径就分别是:
   /home/gzh/.bashrc
   /home/gzh/.vimrc
   ```

   操作流程为：

   1. 先用vim打开xshell配置文件

      ```c
      vim ~/.bashrc
      ```

   2. 通过vim命令G将光标移动到最后一行，在下面插入需要添加的命令，例如我添加的：

      ```c
      # 设置用户名与当前目录颜色，并隐藏主机名——用户名蓝色，当前目录绿色
      export PS1="\[\033[34m\]\u\[\033[0m\]:\[\033[32m\]\W\[\033[0m\]\$ "
      
      # 设置 man 手册默认语言为中文——但是多数函数还是没有中文版介绍
      export LC_MESSAGES=zh_CN.UTF-8
      export MANLANG=zh_CN
      
      # 自定义别名——输入llh即可查看当前目录所有文件的所有信息
      alias llh='ls -alF -i -h'
      ```

      然后退出插入模式，输入:wq保存并退出

   3. 使命令文件生效

      ```c
      source ~/.bashrc
      ```

   vim中同理，想自定义宏、更改vim界面主题颜色的可以修改配置文件以达到效果，具体不会问豆包或者找我。
